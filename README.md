## Getting Started

First, run the development server:

```bash
yarn dev
```

## Структура json объектов:
- начиная с 1 - категории покупок 
- начиная с 100 - категории личных финансовых целей
- начиная с 1000 - клиентские id
- [с 1 - 10] 4 цифры - id карты клиента
